# Taco's File Browser

lightweight SSH file browser made with Tauri / NextJs

![image](doc/img/rm1.png)
![image](doc/img/rm2.png)

## Building

```shell
git clone https://github.com/Tacotakedown/file-browser.git && cd file-browser

bun i

# Dev server
bun tauri dev

# Production Build
bun tauri build

```
